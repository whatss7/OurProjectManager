# POST /api/projects 创建新项目
需要登录才能创建项目。

## 请求头
需要在 Authorization 头信息中包含 token。

## 请求体
一个 Project 对象，其实只需要使用其中的 name 字段和 description 字段。

## 应答
### 创建项目成功
HTTP 201 Created

返回创建的 Project 对象。

### 无 token 或 token 无效
HTTP 401 Unauthorized

返回一个 ApiResponse。

```json
{
    "type": "NotLogin",
    "message": "..."
}
```

只有本项目 Admin 以上才能取消邀请。

## 请求头
需要在 Authorization 头信息中包含 token。

## 应答
### 取消邀请成功
HTTP 200 OK

### 邀请不存在
HTTP 404 Not Found

```json
{
    "type": "InvitationNotFound",
    "message": "..."
}
```

### 无 token 或 token 无效
